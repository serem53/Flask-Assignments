## How to Run

Follow the instructions below to run each Flask project for example to clone 1st_project:

1. Clone the repository:

    ```bash
    git clone https://github.com/serem53/Flask-Assignments/tree/intermediate_flask_topics/1st_project.git
    ```

2. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. Run the Flask app:

    ```bash
    1st_project.py
    ```

Access the projects in your web browser at [http://localhost:5000/](http://localhost:5000/).
